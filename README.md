# Tour de Cicle Indoor — Etapa FTP 20'

Site oficial de consulta de resultados da etapa FTP 20 minutos do Tour de Cicle Indoor, patrocinada por Track & Field.

## Como funciona

1. O participante assiste o vídeo da etapa (obrigatório).
2. Depois libera o acesso ao chat de consulta.
3. O mascote **Cicle** responde perguntas sobre os 78 atletas: nome, academia, categoria, peso, FTP 20 e FTPr.

## Stack

- HTML / CSS / JavaScript estático
- Sem backend, sem banco de dados
- Hospedado no GitHub Pages
